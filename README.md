# House Price Analysis using Tableau

This project analyzes housing market trends using Tableau dashboards.

## Features
- Data preprocessing
- Interactive dashboards
- Top N filters
- House price distribution analysis
- Renovation impact analysis

## Tools Used
- Tableau
- Data Visualization
- GitHub

## Dashboard Insights
- Houses with more bedrooms and bathrooms show higher values
- Renovation influences property value
- Most houses fall in mid-range price segments
